# Proyecto: Control de motores con ESP32 y pantalla Nextion

## Descripción general
Este proyecto implementa el control de motores utilizando una **ESP32**, integrando una **interfaz HMI** desarrollada en **Nextion** para la visualización y manipulación de parámetros del sistema.  
El repositorio incluye el código fuente para el microcontrolador, las plantillas de la pantalla y un enlace al mockup de la interfaz.

---

## Características principales
- Control de motores (dirección, velocidad y función *home*).  
- Comunicación serial entre **ESP32** y **Nextion**.  
- Interfaz HMI intuitiva para usuario final.  
- Código modular y documentado.  
- Simulación de interfaz mediante **mockup interactivo**.

---

Requisitos

Hardware:
- ESP32 (cualquier modelo compatible con UART).
- Pantalla Nextion (modelo básico o avanzado según interfaz).
- Motor DC / paso a paso + driver correspondiente.

Software:
- Micropython / Espressif
- Librería Nextion para ESP32.
- Nextion Editor para editar las plantillas HMI.
